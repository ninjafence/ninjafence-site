# NinjaFence Site

This repository contains the source for the NinjaFence GitHub Pages website.


It includes:

- A landing page for NinjaFence and NinjaDCC
- Product pages for NinjaFence-G1, NinjaFence-G2, and NinjaDCC
- A lightweight docs section for NinjaFence Motion and NinjaDCC
- Static assets such as logos, icons, and product images

## Structure

- `index.html` - homepage
- `style.css` - shared main site styles
- `products/` - product detail pages
- `blog/` - blog and build log pages
- `docs/` - documentation pages
- `assets/` - images, logos, and icons

## Local Preview

Since this is a static site, you can open `index.html` directly in a browser or serve the folder with any simple static server.

## Deployment

This site is intended to be published with GitHub Pages.
