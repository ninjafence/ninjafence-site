About

Hi, I'm Ninja, an embedded software engineer with over 20 years of experience building real-world systems.

I've always loved woodworking. My father is a master carpenter with more than 40 years of experience, and many of the woodworking tools I use today came from him. He is still actively working in the workshop.

The ideas behind my projects come directly from real problems we face in daily woodworking. I started building these tools to help make workshop work more efficient and more comfortable.

If these builds can also help other woodworkers, that would mean a lot to me.

Workshop project

NinjaFence-Motion-G1

NinjaFence-Motion-G1 is a motorized fence positioning system for table saws. It accepts a target position input and drives a stepper-based mechanism to move the fence to the specified location with high repeatability. The design focuses on a single core function: precise, reliable fence positioning, reducing manual adjustment and improving workflow consistency.

NinjaDCC

NinjaDCC (Dust Collection Control) is a wireless system that automatically turns your dust collector on and off based on machine activity. It supports both one-to-one and multi-machine setups, with reliable RF communication and simple pairing for easy workshop integration.
